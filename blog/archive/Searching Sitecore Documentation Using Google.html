Searching Sitecore Documentation Using Google

  • 09 Jul 2013

  • Mike

This post has been migrated from www.experimentsincode.com, we apologise if some of the images or content is missing

Google has a handy tool that allows you to filter results by a specific URL so that only results beneath that URL returned. This gives us a hand way to search the Sitecore documentation: The following sections show the different search strings to use with links to Google for each:

Sitecore 7

 site:sdn.sitecore.net/upload/sitecore7/

Sitecore 6

 site:sdn.sitecore.net/upload/sitecore6/

Sitecore 6.6

 site:sdn.sitecore.net/upload/sitecore6/66/

Sitecore 6.5

 site:sdn.sitecore.net/upload/sitecore6/65/

Sitecore 6.4

 site:sdn.sitecore.net/upload/sitecore6/64/

Sitecore 6.3

 site:sdn.sitecore.net/upload/sitecore6/63/

Sitecore 6.2

 site:sdn.sitecore.net/upload/sitecore6/62/

Sitecore 6.1

 site:sdn.sitecore.net/upload/sitecore6/61/

Sitecore 6.0

 site:sdn.sitecore.net/upload/sitecore6/60/
For some reason Google has indexed all the documents for Sitecore 7 at the moment.

Blog - V4 Released

<< Back to Blogs

  • 01 May 2015

  • Mike

Today is a very big day in the history of Glass.Mapper.Sc with the release of V4! This is the biggest change to Glass.Mapper.Sc in the last 2 years and sees many great new features being added to the framework:

  • Caching
  • Edit Frame
  • IOC removal
  • Sitecore Version specific assemblies
  • MVC improvements
  • Fluent configuration Maps


To find out about these great features please read our early blog post Glass.Mapper.Sc V4. 

It is important to note that the following Nuget packages will be removed from Nuget search listings, this means that you can still get them is you explicitly request them using the Nuget Package Manager Console but they won't appear in Nuget searches:

  • Glass.Mapper.Sc.Windsor
  • Glass.Mapper.Sc.Mvc-v3
  • Glass.Mapper.Sc.Mvc-v4
  • Glass.Mapper.Sc.Mvc-v5
  • Glass.Mapper.Sc.Razor
  • Glass.Mapper.Sc.Razor.CastleWindsor

We have added to our documentation section information on Upgrading from V3 to V4 for those of you already using Glass.Mapper.Sc. 

For those who are still using V3 and not quite ready to take advantage of the awesome new features, never fear! V3 will still be maintained for a little while longer yet. If you are wanting to get future bug releases for V3 versions please see the V3 Assemblies page.

Special thanks should go to Nathanael Mann who has contributed and worked on many of the new features in V4 and to the Glass.Mapper community for supporting the project!
